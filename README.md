# Sunucu Kurulum El Kitabı

Production-ready Linux sunucu kurulumu için kapsamlı rehber.

## 📖 Dokümantasyon

**Canlı Site:** [https://slymanmrcan.github.io/production-handbook/](https://slymanmrcan.github.io/production-handbook/)

## 🎯 Özellikler

- **Güvenlik Odaklı**: SSH hardening, Firewall, Fail2ban
- **Otomasyon**: Hazır scriptler ve konfigürasyonlar
- **Operasyonel**: Runbook'lar ve troubleshooting rehberleri
- **Modern Tasarım**: Dark mode, glassmorphism UI

## 🚀 Hızlı Başlangıç

```bash
# Repoyu klonla
git clone https://github.com/slymanmrcan/production-handbook
cd server-guide

# MkDocs'u kur
pip install mkdocs-material

# Lokal olarak çalıştır
mkdocs serve
```

Tarayıcıda `http://127.0.0.1:8000` adresini açın.
